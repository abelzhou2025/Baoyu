# 双击版使用说明（小白版）

## 你先知道两件事

1. 以后只需要双击这个文件：
   - `/Users/abelzhou/Desktop/Baoyu/一键图文解读.command`
2. 第一次生成图片时，可能会弹浏览器让你登录 Google（Gemini Web 需要）。

## 怎么用

1. 在 Finder 打开 `/Users/abelzhou/Desktop/Baoyu`
2. 双击 `一键图文解读.command`
3. 终端会问你：
   - 链接 URL
   - 内容类型（博客/长文/视频）
   - 输出类型（解读文/信息图/XHS）
   - 发布渠道（微信/X/不发布）
4. 等它跑完，会自动打开结果目录。

## 结果在哪里

每次结果都在：
- `/Users/abelzhou/Desktop/Baoyu/output/时间戳/`

重点看两个文件：
- `draft.md`：图文解读草稿
- `cover.png` 或 `infographic.png`：自动生成图片

## 常见问题

1. 提示 `缺少命令 npx`
   - 说明没装 Node.js。安装 Node.js LTS 后重试。
2. 图片生成失败
   - 常见是首次登录/网络问题。草稿仍然会保留在 `draft.md`。
3. 链接抓不到
   - 部分网站有登录或反爬，可以先手动复制正文到 markdown 再跑管线。
